# Rock-paper-scissor
A simple and interactive Rock Paper Scissors game built using HTML, CSS, and JavaScript. The game allows users to play against the computer, keeps track of scores using localStorage, and displays the result instantly with move icons.
Features
- Play against computer
- Random move generation
- Live score tracking
- Data stored using localStorage
- Reset score button

## Tech Stack
- HTML
- CSS
- JavaScript

## Concepts Used
- DOM Manipulation
- Functions
- Event Handling
- localStorage
- JSON
